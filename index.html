What I Learnt Using SAP Analytics Cloud (SAC)

Reflection on learning SAP Analytics Cloud, making dashboards, and following SAP tutorials.

Getting Started with SAC

I began by following the official Exploring SAP Analytics Cloud tutorial. It guided me through the interface, menus, and how to connect data. At first it seemed complex, but step-by-step tasks built my confidence quickly.

Creating Dashboards

The main skill I practised was building dashboards that tell a clear story. I focused on picking the right KPIs and arranging visuals for quick understanding.

My workflow

  1. Connect or import a dataset.
  2. Choose essential KPIs.
  3. Select visuals (bar, line, map, KPI cards).
  4. Lay out charts in a logical flow.
  5. Test readability with others.

Skills I Developed

  • Data visualisation: turning raw numbers into clear charts.
  • Analytical thinking: selecting the right KPIs and metrics.
  • Practical application: applying tutorials to real tasks.
  • Communication: writing captions and hover text that explain the ‘why’.

Reflection

Learning SAP Analytics Cloud showed me how analytics supports decision‑making. I realised dashboard design is about clarity, not decoration. This experience motivates me to continue building skills in SAC and other BI tools.

Tutorials & Resources I Used

  • Exploring SAP Analytics Cloud — step‑by‑step guidance on models, stories, and dashboards.
  • SAP Help Guide: Working with Models — in‑depth documentation on creating and managing data models.
  • SAP Analytics Cloud Course — additional structured lessons and resources.
